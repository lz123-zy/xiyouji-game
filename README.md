# 🐵 西游记·祸起观音院

一款基于 Python + Pygame 的 2D 俯视角 RPG 小游戏，以中国古典名著《西游记》为题材。

玩家操控**齐天大圣孙悟空**，在村庄、郊外、观音院三个场景中探索，与 NPC 对话获取任务，击败妖怪，最终击败 Boss **牛魔王**，解除观音院危机。

---

## 🎮 游戏特色

- **三场景探索**：村庄 → 郊外 → 观音院，场景无缝切换
- **NPC 对话系统**：8 位个性 NPC，对话随任务进度变化
- **怪物 AI**：巡逻 → 追击 → 返回，智能状态机驱动
- **回合制战斗**：攻击动画、受击闪红、魔法特效一应俱成
- **Boss 战**：牛魔王含狂暴阶段（50% 血量触发），挑战十足
- **完整剧情**：接取任务 → 探索战斗 → 击败 Boss → 凯旋复命

## 🚀 快速开始

### 环境要求

- Python 3.10+
- pip

### 安装与运行

```bash
# 1. 克隆仓库
git clone https://github.com/lz123-zy/xiyouji-game.git
cd xiyouji-game

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动游戏
python main.py
```

> 依赖：`pygame`、`pytmx`

## 🎹 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `A` / `S` / `D` 或 方向键 | 移动角色 |
| `E` / 空格 | 与 NPC 对话 / 场景交互 |
| `J` | 战斗中攻击 |
| `R` | 战斗失败后重新挑战 |
| `P` | 暂停 / 继续 |
| `H` | 查看帮助 |
| `M` | 静音 / 取消静音 |
| `F11` / `Alt+Enter` | 全屏切换 |
| `Esc` | 关闭对话 / 退出游戏 |

- **小怪 ×4**：巡逻于寺院各处，触发战斗后进入回合制对决

## 🏗️ 项目结构

```
西游记小游戏/
├── main.py                  # 入口文件
├── requirements.txt         # Python 依赖
├── src/                     # 源代码
│   ├── game.py              # 主游戏控制器（事件循环、状态机、场景管理）
│   ├── player.py            # 玩家角色（四向动画、碰撞检测）
│   ├── npc.py               # NPC 实体（对话、动画）
│   ├── monster.py           # 怪物 AI（巡逻/追击/返回状态机）
│   ├── boss.py              # Boss 子类（牛魔王，含狂暴机制）
│   ├── battle.py            # 回合制战斗系统
│   ├── quest.py             # 任务状态机（5 阶段）
│   ├── scene.py             # 场景管理器
│   ├── tmx_map.py           # TMX 地图加载器
│   ├── camera.py            # 摄像机跟随系统
│   ├── dialog.py            # 对话框渲染器
│   ├── ui.py                # UI 界面（开始、帮助、暂停、结果）
│   ├── audio.py             # 音频管理器
│   ├── animation.py         # 帧动画控制器
│   └── settings.py          # 全局配置常量
├── data/                    # 数据配置
│   ├── dialogs.json         # 对话树
│   ├── monsters.json        # 怪物属性
│   └── scenes.json          # 场景元数据
├── resource/                # 游戏资源
│   ├── tmx/                 # Tiled 地图文件
│   ├── img/                 # 图片素材（角色、怪物、UI、场景）
│   ├── font/                # 中文字体
│   └── sound/               # 音频文件
└── 项目需求分析报告.docx       # 需求分析文档
```
