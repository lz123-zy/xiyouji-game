# 🐵 西游记·祸起观音院

一款基于 Python + Pygame 的 2D 俯视角 RPG 小游戏，以中国古典名著《西游记》为题材。

玩家操控**齐天大圣孙悟空**，在村庄、郊外、观音院三个场景中探索，与 NPC 对话接取降妖任务，击败四只妖怪，最终迎战 Boss **牛魔王**，解除观音院危机，凯旋复命。

---

## 游戏特色

- **三场景无缝切换**：村庄 → 郊外 → 观音院，场景按任务阶段控制通行
- **NPC 对话系统**：村庄 5 位、郊外 2 位 NPC，对话随任务进度动态变化
- **怪物 AI 巡逻**：三状态机驱动（巡逻 → 追击 → 返回），支持视野检测与牵引距离
- **回合制战斗**：攻击帧动画、受击闪红、魔法特效、动态血条
- **Boss 狂暴机制**：牛魔王 HP 低于 50% 时攻击力与攻速大幅提升
- **完整主线剧情**：接取任务 → 探索战斗 → 击败 Boss → 叙事画面 → 回村复命

## 快速开始

### 环境要求

- Python 3.10+
- pip

### 安装与运行

```bash
# 1. 克隆仓库
git clone https://github.com/lz123-zy/xiyouji-game.git
cd xiyouji-game

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动游戏
python main.py
```

> 依赖库：`pygame`、`pytmx`

## 核心设计

### 任务状态机（`quest.py`）

```
NOT_ACCEPTED → ACCEPTED → CLEARED → RETURNED → COMPLETE
  (未接取)      (已接取)    (已清怪)    (已回村)    (完成)
```

场景切换由任务阶段控制：村庄到郊外需 `ACCEPTED`，观音院回郊外需 `CLEARED`，郊外回村需 `CLEARED`，避免玩家在任务未完成时误触发。

### 怪物 AI（`monster.py`）

三状态机在地图探索时持续运行：

- **巡逻**：在出生点附近随机走动，碰墙自动换向
- **追击**：视野内发现玩家后追踪，设有最大牵引距离防止跑出地图
- **返回**：丢失目标后返回出生点，恢复巡逻

### Boss 狂暴机制（`boss.py`）

继承 `Monster`，覆写数值：HP 归零进入 `die` 动画前，若 HP ≤ 50% 则触发 `_enter_enraged_phase()`，攻击力从 16 提升至 24，攻击间隔从 1.5s 缩短至 1.0s。
